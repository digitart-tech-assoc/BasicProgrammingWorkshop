### Q8[B]. $`Sqrt`$

任意の変数numの平方根(２乗するとnumになる数)を求めます。resultを0から0.001ずつ増やしながら「しらみつぶし」に探してください。resultを２乗した値がnumを超えた瞬間にループを終了させなさい。

```javascript
let num = 2;
let result = 0;

// ここにコードを入力

console.log(num + "の平方はだいたい" + result + "です。");
```

<details><summary>入力と出力</summary>

入出力例1
```
[in]
num = 2
result = 0

[out]
2の平方はだいたい1.415です。
```

入出力例2
```
[in]
num = 1000
result = 0

[out]
1000の平方はだいたい31.623です。
```
</details>
